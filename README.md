# Professional Portfolio Website

## 📋 Description
A modern, responsive portfolio website built with HTML, CSS, JavaScript, and Bootstrap 5.

## ✨ Features

✅ **Responsive Navigation** - Sticky header with mobile-friendly menu  
✅ **Hero Section** - Eye-catching intro with animated background  
✅ **About Section** - Professional bio and skill badges  
✅ **Services** - Display your services/expertise  
✅ **Portfolio** - Showcase your projects  
✅ **Contact Form** - Functional contact form with validation  
✅ **Social Links** - Connect to your social media  
✅ **Smooth Animations** - Interactive scroll animations  
✅ **Fully Responsive** - Mobile, tablet, and desktop support  

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom styling and animations
- **JavaScript (ES6+)** - Interactive features
- **Bootstrap 5** - Responsive grid system
- **Font Awesome** - Icon library

## 📂 Project Structure


## 🚀 Features Explained

### Hero Section
- Full-screen introduction with animated background
- Call-to-action buttons
- Social media links

### About Section
- Professional profile with image
- Skills badges
- CV download link

### Services Section
- 4 service cards with icons
- Hover animations
- Professional descriptions

### Portfolio Section
- Project showcase
- Hover effects on project cards
- Links to view projects

- Contact information (location, email, phone)
- Functional contact form
- Form validation
- Success message on submission

## 📝 How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content with your own information
4. Add your images and project links
5. Deploy to your hosting service

## 🎨 Customization

Edit the following files to customize:

- **index.html** - Change content, text, and structure
- **portfolio.css** - Modify colors, fonts, and layouts
- **portfolio.js** - Update functionality

### Color Scheme
The main colors are defined in `portfolio.css`:
```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #f39c12;
    --text-color: #333;
    --light-bg: #ecf0f1;
}
```

## 📱 Responsive Design

The portfolio is optimized for all screen sizes:
- **Desktop** - Full layout
- **Tablet** - Adjusted spacing and font sizes
- **Mobile** - Stacked layout with hamburger menu
